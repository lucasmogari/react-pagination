# react-pagination

## DEPRECATED

Headless react hook for pagination.

- Headless hook
- Small bundle
- Typescript

Bundle size:
3.4 kB (minified)
1.4 kB (minified + gzipped)


### Install

```
npm install @lucasmogari/react-pagination --save
# or 
yarn add @lucasmogari/react-pagination
```
